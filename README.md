# Семантический анализ образа России в СМИ

** Описание проекта **

Программа предназначена для автоматизированного сбора и анализа публикаций российских СМИ, посвящённых образу России в медиапространстве.
Может быть адаптирована для анализа любой тематики путем изменения списка ключевых слов и RSS-источников без изменения основной логики работы.

** Функционал программы:**

* сбор новостей из RSS-лент;
* очистка и лемматизация текстов;
* частотный анализ;
* построение облака слов;
* построение аналитических графиков.

** Используемые языки **

Основной язык: Python

Вспомогательный язык: R

** Установка зависимостей **

Python

```bash
pip install -r requirements.txt
```

R

```r
install.packages("ggplot2")
```

** Запуск программы **

```bash
python start.py
```

** Результаты **

Таблицы сохраняются в:


Графики сохраняются в:
