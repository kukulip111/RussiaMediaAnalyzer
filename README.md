# Семантический анализ образа России в СМИ

## Описание проекта

Программа предназначена для автоматизированного сбора и анализа публикаций российских СМИ, посвящённых образу России в медиапространстве.

Программа может быть адаптирована для анализа любой тематики путем изменения списка ключевых слов и RSS-источников без изменения основной логики работы.

## Ограничения и особенности работы программы 
Перед использованием программы рекомендуется ознакомиться со 
следующими особенностями ее работы: 

1) В текущей версии программы анализируются публикации только 
из следующих средств массовой информации: РИА Новости; РБК; Коммерсантъ. 

2) Отбор публикаций осуществляется по заранее заданному списку 
ключевых слов, связанному с образом России. Если публикация не 
содержит указанных ключевых слов, она не будет включена в 
анализ. 

3) Пользователь может адаптировать программу для исследования 
другой тематики. Для этого необходимо изменить: 

- список ключевых слов в файле scripts/parser.py;

- список RSS-источников в словаре RSS_FEEDS.

4) Результаты анализа зависят от доступности RSS-лент и объема 
найденных публикаций. При небольшом количестве публикаций 
результаты могут быть менее репрезентативными. Также, как 
правило, RSS-ленты содержат новости за последние несколько 
дней или недель, поэтому программа не предназначена для 
анализа исторических архивов СМИ за длительный период 
времени. 


## Функционал программы

* сбор новостей из RSS-лент;
* очистка и лемматизация текстов;
* частотный анализ;
* построение облака слов;
* построение аналитических графиков.
